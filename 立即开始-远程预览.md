# 🚀 立即开始：远程预览（3 步搞定）

## ✅ 已完成的配置

我已经为你配置好了所有内容，现在只需要运行一个命令！

---

## 📱 第一步：启动远程预览

在项目目录打开 PowerShell，运行：

```powershell
npm run dev:cloudflare
```

**就这么简单！** 首次运行会自动下载工具，等待 1-2 分钟。

---

## 🔗 第二步：获取公网链接

运行后，在输出中找到类似这样的 URL：

```
https://xxxx-xxxx-xxxx.trycloudflare.com
```

**复制这个链接！**

---

## 📲 第三步：在 iPhone 上打开

1. 打开 iPhone Safari 浏览器
2. 粘贴刚才复制的链接
3. 即可访问！

**无需同一 Wi-Fi，在任何地方都可以访问！**

---

## 💡 其他命令

```powershell
# 自动选择工具（推荐）
npm run dev:tunnel

# 使用 ngrok（需要先安装）
npm run dev:ngrok

# 预览生产版本
npm run preview:tunnel
```

---
