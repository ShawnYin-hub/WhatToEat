# 🆓 免费地图服务方案 - OpenStreetMap

## ✨ 完全免费，无需绑卡！

应用现在支持 **OpenStreetMap (OSM)** 作为免费的地图服务选项。

## 🌍 OpenStreetMap 简介

- **完全免费**：开源项目，无任何费用
- **无需绑卡**：不需要信用卡
- **无需 API Key**：不需要注册或配置
- **全球覆盖**：覆盖全球范围
- **数据丰富**：包含大量餐厅、咖啡厅等POI数据

## 🎯 适用场景

- ✅ 海外用户（特别是欧美地区）
- ✅ 不想绑定信用卡的用户
- ✅ 个人项目或学习项目
- ✅ 数据需求不是特别精细的场景

## 📊 数据特点

### 优势
- ✅ 完全免费
- ✅ 全球覆盖
- ✅ 数据更新频繁（社区维护）
- ✅ 开放数据，无使用限制

### 限制
- ⚠️ 数据详细程度可能不如商业API
- ⚠️ 某些地区数据可能不完整
- ⚠️ 不提供评分、价格等商业信息
- ⚠️ 查询速度可能略慢

## 🚀 如何使用

1. 在应用中打开"地图服务"选择器
2. 选择"**OpenStreetMap**"选项
3. 会显示绿色提示："✓ 完全免费，无需配置"
4. 直接使用，无需任何配置！

## 🔍 支持的功能

- ✅ 地点搜索（使用 Nominatim API）
- ✅ 附近餐厅搜索（使用 Overpass API）
- ✅ 自动定位
- ✅ 手动输入地址
- ✅ 搜索范围设置
- ✅ 关键词过滤

## 📝 技术细节

### 使用的 API

1. **Nominatim API**
   - 用于地点搜索和地理编码
   - 官方实例：`https://nominatim.openstreetmap.org/`
   - 请求限制：每秒1次（请合理使用）

2. **Overpass API**
   - 用于查询POI数据（餐厅、咖啡厅等）
   - 官方实例：`https://overpass-api.de/api/interpreter`
   - 支持查询餐厅、咖啡厅、酒吧等

### 数据格式

OSM 数据格式与高德/Google 不同，但应用已自动处理，用户体验一致。

## ⚠️ 使用注意事项

1. **请求频率限制**
   - Nominatim 有使用限制（建议每秒1次）
   - 不要频繁发送请求
   - 应用已实现防抖，避免过度请求

2. **数据准确性**
   - OSM 数据由社区维护
   - 不同地区数据质量可能不同
   - 如果某个地区数据不完整，可以切换到其他服务

3. **User-Agent**
   - OSM API 要求设置 User-Agent
   - 应用已自动设置
