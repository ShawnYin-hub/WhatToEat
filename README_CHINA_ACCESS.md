# 🇨🇳 中国用户访问说明

## ✅ 中国用户推荐使用：高德地图

### 为什么？

1. **访问速度快**
   - 高德地图服务器在中国
   - 无需 VPN，直接访问
   - 响应速度快

2. **数据准确**
   - 专为中国地区优化
   - 覆盖全面，数据详细
   - 包含评分、电话等信息

3. **完全免费**
   - API Key 已内置
   - 无需配置
   - 无需绑卡

4. **功能完整**
   - 定位准确
   - 搜索快速
   - 导航方便

---

## ⚠️ OpenStreetMap 在中国的情况

### 访问限制

OpenStreetMap 的 API 服务器位于海外：
- `nominatim.openstreetmap.org`
- `overpass-api.de`

**在中国内地可能：**
- ❌ 无法访问
- ⚠️ 访问很慢
- ⚠️ 需要 VPN 才能正常使用

即使部署到 Vercel：
- Vercel 服务器可以访问 OpenStreetMap API
- 但中国用户访问 Vercel 网站本身可能受限
- OpenStreetMap API 调用可能失败

---

## 🎯 使用建议

### 中国用户：
1. **默认选择：高德地图** ✅
   - 点击地图服务选择器
   - 选择"高德地图"
   - 所有功能都可以正常使用

2. **如果高德地图数据不足：**
   - 可以尝试 OpenStreetMap
   - 可能需要 VPN
   - 数据可能不完整

### 海外用户：
1. **推荐：OpenStreetMap** ✅
   - 完全免费
   - 不需要 API Key
   - 全球覆盖

2. **或使用：Google Maps**
   - 需要 API Key
   - 数据更详细
