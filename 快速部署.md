# 🚀 快速部署到 Vercel（3步完成）

## 最简单的方法：通过 GitHub

### 第 1 步：上传代码到 GitHub

**选项 A：使用 GitHub Desktop（推荐给新手）**

1. 下载：https://desktop.github.com/
2. 安装后打开，登录 GitHub 账户
3. File > Add Local Repository
4. 选择：`C:\Users\30449\what-to-eat-today`
5. 点击 "Publish repository"
6. 填写仓库名：`what-to-eat-today`
7. 点击发布

**选项 B：使用网页上传**

1. 访问 https://github.com/new
2. 创建新仓库：`what-to-eat-today`
3. 下载项目文件夹中的所有文件
4. 在 GitHub 网页点击 "uploading an existing file"
5. 拖拽所有文件上传
6. 点击 "Commit changes"

### 第 2 步：在 Vercel 部署

1. 访问：https://vercel.com
2. 点击 "Sign Up"（用 GitHub 登录）
3. 点击 "Add New..." > "Project"
4. 找到 `what-to-eat-today`，点击 "Import"
5. 直接点击 "Deploy"（使用默认设置）
6. 等待 1-2 分钟

### 第 3 步：获得网址

部署完成后，您会看到：
```
✅ Deployment successful!
🌍 https://what-to-eat-today.vercel.app
```

**这个网址就是您的网站！** 可以分享给任何人使用。

## ❓ 常见问题

**Q: 需要付费吗？**
A: 不需要，Vercel 免费套餐足够使用。

**Q: 需要绑卡吗？**
A: 不需要，完全免费。

**Q: 网址可以自定义吗？**
A: 可以，在 Vercel 设置中添加自己的域名（需要购买域名）。

**Q: 部署失败怎么办？**
A: 检查 `package.json` 是否正确，或者查看 Vercel 的部署日志。
