# 🔧 Nominatim API 403 错误解决方案

## ⚠️ 问题

Nominatim API 返回 403 Forbidden，说明 API 拒绝了请求。

## 🔍 可能的原因

1. **User-Agent 问题**
   - User-Agent 不符合要求
   - 或者 User-Agent 格式不正确

2. **使用政策限制**
   - Nominatim API 有严格的使用政策
   - 可能触发了某些限制

3. **请求频率限制**
   - 请求过于频繁
   - 触发了频率限制

## ✅ 解决方案

### 方案 1：使用高德地图（推荐，最简单）

由于 OpenStreetMap 的 API 限制，对于中国用户：

1. **使用高德地图**
   - 数据更准确
   - 访问速度快
   - 不需要 VPN
   - 完全免费

2. **在应用中选择高德地图**
   - 打开应用
   - 选择"高德地图"服务
   - 所有功能都可以正常使用

### 方案 2：优化 Nominatim 请求（如果必须使用）

如果需要继续使用 OpenStreetMap：

1. **添加更详细的错误处理**
   - 查看 Nominatim API 的错误响应
   - 了解具体的拒绝原因

2. **调整 User-Agent**
   - 使用更标准的格式
   - 添加联系信息

3. **添加请求延迟**
   - 避免请求过于频繁
   - 遵守使用政策
