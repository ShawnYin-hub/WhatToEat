# 🚀 快速部署到 Vercel（3步完成）

## 最简单的方法：GitHub Desktop + Vercel

### 第 1 步：上传代码到 GitHub（5分钟）

1. **下载 GitHub Desktop**
   - 访问：https://desktop.github.com/
   - 下载并安装

2. **打开 GitHub Desktop 并登录**
   - 使用您的 GitHub 账户登录（如果没有，先注册：https://github.com）

3. **添加项目**
   - 点击 "File" > "Add Local Repository"
   - 点击 "Choose..." 
   - 选择文件夹：`C:\Users\30449\what-to-eat-today`
   - 点击 "Add Repository"

4. **发布到 GitHub**
   - 点击工具栏的 "Publish repository" 按钮
   - 仓库名称：`what-to-eat-today`
   - 可以选择 "Keep this code private"（私有）或不选（公开）
   - 点击 "Publish Repository"
   - 等待上传完成（通常1-2分钟）

### 第 2 步：在 Vercel 部署（3分钟）

1. **访问 Vercel**
   - 打开：https://vercel.com
   - 点击 "Sign Up"

2. **登录**
   - 选择 "Continue with GitHub"
   - 授权 Vercel 访问您的 GitHub

3. **导入项目**
   - 登录后，点击 "Add New..." > "Project"
   - 找到您的仓库 `what-to-eat-today`
   - 点击 "Import"

4. **部署**
   - 直接点击 "Deploy"（使用默认设置即可）
   - 等待 1-2 分钟自动部署

### 第 3 步：获得网址 ✅

部署完成后，您会看到：
```
✅ Deployment successful!
🌍 https://what-to-eat-today.vercel.app
```

**这个网址就是您的网站！可以分享给任何人使用！**

## 方法二：使用命令行（适合熟悉命令行的用户）

### 如果有 GitHub 账户：

```bash
cd C:\Users\30449\what-to-eat-today

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit"

# 创建 GitHub 仓库后，连接并推送（替换 YOUR_USERNAME）
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/what-to-eat-today.git
git push -u origin main

# 然后在 Vercel 导入项目（同方法一的第2步）
```

### 如果使用 Vercel CLI：

```bash
# 安装 Vercel CLI
npm install -g vercel

# 登录
vercel login

# 部署
cd C:\Users\30449\what-to-eat-today
vercel

# 按照提示操作，等待部署完成
```
