## 🎯 功能说明

### 高德地图（中国用户）
- 点击"带我导航" → 打开高德地图导航
- 支持国内地址导航

### OpenStreetMap（海外用户）
- 点击"带我导航" → 打开 Google Maps 导航
- 支持全球地址导航
- 即使没有 Google Maps API Key 也可以使用（只是打开地图显示位置）

### 复制地址（备选方案）
- 如果无法生成导航 URL，显示"复制地址"按钮
- 点击后复制餐厅名称、地址和坐标
- 用户可以粘贴到任何地图应用搜索
